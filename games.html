Untitled Wind Waker Rip Off

This is my latest project. It builds heavily off of the lessons I learned from making my previous game. It is a first person narrative adventure game with trading mechanics. With this project I wanted to flex my shader skills as well as my ability to craft a dynamic narrative. It will be a fairly short, highly replayable game with unique routes through the same environment. I will have an early playable demo available mid July.

Technologies used: Unity, C#, Shaderlab, HLSL, Fungus, Blender, Gimp, Ableton
