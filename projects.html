Tour Finder

This is a single page web tool I designed to make booking a tour easier for small bands. When you search a city a list of venues and bands local to that area populates. Each listing has a link to their respective websites. I wanted to make this because when I was in a band, booking tours were the least fun part of the whole experience.

Technologies used: HTML, CSS, Javascript, React, Node, Express, MongoDB, Google Maps Geocoding API
