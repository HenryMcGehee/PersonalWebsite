Untitled Spirit Game


This project is currently shelved. It built heavily off of the lessons learned from making my previous game. It is a first person narrative adventure game with trading mechanics. With this project I wanted to flex my shader skills as well as my ability to craft a dynamic narrative. It will be a fairly short, highly replayable game with unique routes through the same environment.

Technologies used: Unity, C#, Shaderlab, HLSL, Fungus, Blender, Gimp, Ableton
